🔥 需要配置 Firebase

本应用使用 Firebase Firestore 实现实时多人协作。请按以下步骤配置：

  1. 前往 Firebase Console 创建项目
  2. 进入项目 → Firestore Database → 新建数据库（测试模式即可）
  3. 进入项目设置 → 你的应用 → 复制 SDK 配置
  4. 用文本编辑器打开本 HTML 文件，找到 FIREBASE_CONFIG，填入配置
  5. 保存后重新打开即可使用

配置仅需一次，所有成员共用同一个 Firebase 项目
